## Кто я и чем занимаюсь

Разрабатываю AI-решения, которые автоматизируют рутинные процессы в компаниях: обработку заявок, анализ документов, работу с базами знаний, первичный контакт с клиентами и кандидатами.

Проектирую системы на базе современных LLM (OpenAI), интегрирую их с Telegram, Gmail, Google Sheets и внутренними сервисами компании.

Моя задача — не просто подключить ИИ, а встроить его в реальный рабочий процесс так, чтобы он экономил время и снижал нагрузку на сотрудников.

## Какие задачи я решаю

- Автоматическая обработка входящих заявок и резюме
- Создание AI-ассистентов для Telegram
- Построение RAG-систем (ответы на основе базы знаний компании)
- Анализ документов (PDF, текст, изображения)
- Мониторинг и анализ контента конкурентов
- Автоматизация бизнес-логики через n8n
