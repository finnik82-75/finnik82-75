### AI HR Assistant  
Автоматическая обработка резюме.

Система:
- принимает резюме через Gmail  
- извлекает данные из PDF  
- анализирует навыки кандидата  
- сопоставляет с вакансиями  
- сохраняет результаты в Google Sheets  
- отправляет ответ кандидату  

Технологии: n8n, OpenAI API, Gmail API  

https://github.com/finnik82-75/ai-hr-assistant

### AI Competitor Monitoring

- формирование структурированных аналитических выводов  

Технологии: FastAPI, Selenium, OpenAI API  

https://github.com/finnik82-75/ai-competitor-monitoring
