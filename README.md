**AI-архитектор | Автоматизация процессов | RAG-системы | AI-агенты**

## Кто я и чем занимаюсь

Разрабатываю AI-решения для автоматизации рутины: обработка заявок и резюме, анализ документов, работа с базами знаний, первичный контакт с клиентами и кандидатами. Проектирую системы на LLM (OpenAI), интегрирую с Telegram, Gmail, Google Sheets и внутренними сервисами. Цель — встраивать ИИ в рабочие процессы так, чтобы экономить время и снижать нагрузку на сотрудников.

## Какие задачи я решаю

- Обработка входящих заявок и резюме
- AI-ассистенты для Telegram
- RAG-системы (ответы на основе базы знаний)
- Анализ документов (PDF, текст, изображения)
- Мониторинг и анализ контента конкурентов
- Автоматизация бизнес-логики через n8n

## Технологический стек

**AI и LLM:** OpenAI API, Embeddings, Structured Output, Vision API

**RAG:** ChromaDB, Chunking, Retrieval, контекстная фильтрация

**Автоматизация:** n8n, Webhooks, API-интеграции

**Backend:** Python, FastAPI

**Интеграции:** Gmail API, Google Sheets API, Telegram Bot API
